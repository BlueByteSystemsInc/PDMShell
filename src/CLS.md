# CLS Command
## Description
Clears the current session.

## Syntax
```bash
cls
```

## Parameters
Command has no parameters.

## Examples
```bash
cls
```

## Remarks
- You can alternatively set the Line Limit Count from the settings to remind the session to clear every count of lines.

## Availability
Available since PDMShell 2.0.0.
