# CREATEVIRTUALFROMSOURCE Command Documentation

## DESCRIPTION:
Creates virtual documents from a CSV source.

## SYNTAX:
```bash
createvirtualfromsource -filePath
```

## PARAMETERS:
- `filePath`: CSV source file used to create virtual documents.
