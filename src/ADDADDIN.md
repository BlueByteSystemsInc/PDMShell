# addaddin
## Description
Adds a PDM add-in from a specified directory.

## Syntax

```text
addaddin -Directory
```

## Parameters

| Parameter | Required | Description |
| --- | --- | --- |
| None | No | This command has no parameters. |

### Parameter Details

- `Directory`: Directory that contains the add-in files.

## Availability
Available since PDMShell 2.0.0.
