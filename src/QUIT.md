# QUIT Command Documentation

## DESCRIPTION:
Quits the application.

## SYNTAX:
```bash
quit -silent
```
## PARAMETERS:
- `-silent`:
(Optional) suppresses the close dialog box.

## REMARKS:
- This command runs silently in scripts. Read more about [scripting](/src/scripting.html).
