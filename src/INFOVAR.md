# INFOVAR Command Documentation

## DESCRIPTION:
Gets information about a variable.

## SYNTAX:
```bash
infovar -variableName
```

## PARAMETERS:
- `variableName`: The variable name to retrieve information for.

## EXAMPLES:
infovar -v Description
