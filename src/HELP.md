# HELP Command Documentation

## DESCRIPTION:
Provides help about a command.

## SYNTAX:
```bash
help [-command|-c]
```
## PARAMETERS:
-`command`: The specific command you need help with.

## EXAMPLES:
```bash
help -c cd #opens the help page about the change directory command
```
