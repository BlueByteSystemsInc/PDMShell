# DUMP Command Documentation

## DESCRIPTION:
Dumps all session text into a log file.

## SYNTAX:

```bash
dump filePath
```
## PARAMETERS:
 
- `filePath`: The log file to dump session details into.

## EXAMPLES:
```bash
dump -filePath "sessionlog.txt"
```
